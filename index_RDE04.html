HTML

Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee.

A HTML é utilizada para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo, e que podem ser vinculados através de hyperlinks.

1990 - Criação do HTML
Invenção dA Linguagem HTML.
1995 - Versão 2.0
Formalizar todas as características HTML que já eram utilizadas.
1997 - Versão 3.2
Permite a criação de tabelas, applets e texto flutuante ao redor de imagens.
1999 - Versão 4.01
Suporte para opções multimídia, folhas de estilo e melhorar as práticas de marcação.
2014 - Versão HTML 5
Aperfeiçoar a experiência da World Wide Web para os desenvolvedores e usuários finais.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas.
Sem o uso de tags o navegador exibe um texto plano.
As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

Imagens

Uma representação visual de uma pessoa, objeto ou paisagem. Pode ser:

  • Imagem digital: é uma imagem bidimensional que utiliza de números binários codificados de modo a permitir seu armazenamento, transferência, impressão ou reprodução, e seu processamento por meios eletrônicos.
Formatos de imagem: Padrão de codificação, compressão e definição que determina tipo de arquivo gráfico, exemplo: .BMP, .TIFF, .GIF, . JPEG, .PNG . Programas: Gimp e Photoshop.

No momento de escolher uma imagem da internet é importante ficar atento ao direito de imagem, por isso deve-se usar o Creative Commons. Este consiste em varias licenças públicas que permitem uma dipensão gratuida de obras.

Marcação de listas

As tags de listas definem: listas não-ordenadas, listas ordenadas e lista detalhada.

Lista não-ordenada

Elementos sem ordem pré definida. Sintaxe: inicia com tag ul, e cada item com li.

Lista ordenada

Possui elemento que define a ordem dos elementos. Sintaxe: inicia com tag ol, parâmetro type="elemento" e cada item com li.

Lista detalhada

Permite que elementos da lista possam ser mais bem explicados, como "subelemento". Sintaxe: inicia com tag dt, o primeiro elemento incia e finaliza com dt, o elemento detalhado inicia e finaliza com dd.
