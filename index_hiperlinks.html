Hiperlinks ou links

Hiperlink é uma referência dentro de um documento em HTML, que direciona para outros arquivos.
Estes links podem ser remotos, internos, locias, com endereços de correio e arquivos.

TAGs

Estrutura "target="_blank" é utilizando para abrir o link em uma nova página.

Hiperlink remoto

Referencia sites ou páginas web

Hiperlink interno

Relaciona paragrafos e texto dentro da própria página.

Hiperlink local

Refencia para outras páginas dentro do mesmo site.

Hiperlink em imagem

Relaciona imagem.

Hiperlink para download

Referencia para um arquivo com possibilidade de download.

Media Types; é um identificador padrão usado na internet para relacionar o tipo de dado que um arquivo tem.

Hiperlink para Endereço eletrônico

Referencia para um endereço eletrônico.
